# ユーザー向けかんたん操作ガイド



## 飯蔵について

画像

- 買ってきた食品を忘れてしまい冷蔵庫の奥に賞味期限が切れた〇〇が…
- 買い物にきたけどあれ家にあったっけ？
- めんどくさいけどちゃんと料理したい！



そんな方のために簡単に冷蔵庫の中身を期限付きで管理できるアプリです。



## アカウント作成・ログイン

### 🔰新規登録

1. トップ画面の「始めて使う」ボタンを押します。
2. お名前とパスワードを入力して、「登録」ボタンを押してください。
3. 「ログイン」画面に自動遷移しますので、先ほど入力したお名前とパスワードを入力し、「冷蔵庫を開く」ボタンを押してください。



### ✅既に使っている方

1. トップ画面の「冷蔵庫を開く」ボタンを押します。
2. 登録したお名前とパスワードを入力して画面下の「冷蔵庫を開く」ボタンを押します。
3. 認証が成功したらホーム画面へ遷移します。



### 👋ログアウトしたい時

1. 画面下部のナビバーから「登録情報」をタップし、移動します。

2. 登録情報ページの一番下の「ログアウト」ボタンからログアウトできます。

   

## ナビゲーション

ログイン後、画面下部のナビバーから以下のページに自由に行き来できます。

- ホーム
- 冷蔵庫(登録した食品のリスト)
- レシピ提案
- メモリスト
- 登録情報



## ホーム画面

- 冷蔵庫の中を確認するように登録した食品、数量、賞味期限を確認することができます。
- 期限が近い順と最近登録した食品のカード形式で、最新の3件が表示されます。
- カードを押すとその食品の詳細を見ることができます。
- 「もっと見る」ボタンを押すと登録されている全ての食品が確認できます。下部のナビバーにある「冷蔵庫」と同じページへ遷移します。



### ❇️ホーム画面の便利機能

- 期限が本日までの食品がある場合、画面上部に赤のアラートでお知らせします。
- 期限が過ぎた食品がある場合は「**期限切れをまとめて削除**」ボタンで一括削除ができます。



## 食品を登録する

1. ホーム画面下の「**食品登録**」ボタンを押します。
2. 食品登録フォームにデータを入れてください。※**「食品名」、「カテゴリ」、「数量」、「賞味期限」は必須事項**です。「食品画像」は任意です。



### 🍎プリセット食品リストについて(めんどくさい人向け)

食品登録フォームから食品データページへ遷移できます。

- カテゴリから特定のカテゴリを選択し、「検索」ボタンを押すと絞り込みができます。
- 検索マークの入力欄にキーワードを入れ、「検索」ボタンを押すと食品名で絞り込むことも可能です。
- カテゴリとキーワードの両一致検索も可能です。
- 該当の食品が見つかったら、＋ボタンを押すと食品登録フォームへ自動入力されます。

## 冷蔵庫(食品リスト)

ホーム画面の「もっと見る」ボタンまたはナビバーの「冷蔵庫」を押すと登録されている全ての食品が確認できます。



### 📑ソート機能

「期限が近い順」、「期限が遠い順」、「登録が新しい順」、「登録が古い順」またはカテゴリでソートが可能です。デフォルトは期限が近い順です。

1. ページ上部のソート選択ボタンでどの順に表示したいか選びます。
2. カテゴリ選択ボタンでカテゴリも指定できます。
3. 選択したら三本線の「ソート」マークボタンを押すと、ソートが完了します。





## 食品詳細を見る

- ホーム画面の食品カードまたは「もっと見る」「冷蔵庫」ページで食品を押すと、登録した内容を見ることができます。
- 下部の「ゴミ箱」マークのボタンで食品を削除することができます。
- 閉じたい場合は上部の「×」ボタンを押してください。



## 冷蔵庫メモ

買い物メモ等に使えるメモページです。画面下部のナビバーの「メモ」から遷移できます。

1. 「メモを作成」ボタンを押します
2. 「メモを作成」ページの白の部分のエディターに内容を入力し、「登録」ボタンを押します。
3. 登録されたメモはリスト形式で表示されます。メモを押すと全文を確認できます。
4. 「紙とえんぴつ」マークボタンを押すとメモの編集ができます。「更新」ボタンで更新できます。
5. 「ゴミ箱」マークを押すとメモを削除できます。
